# Microservices Banking Application

Application bancaire basée sur une architecture microservices.

## Services
- Discovery Service (Eureka) - Port 8761
- Config Service - Port 8888
- Gateway Service - Port 8080
- Beneficiaire Service - Port 8081
- Virement Service - Port 8082
- Chatbot Service - Port 8083
- Angular Frontend - Port 4200

## Prérequis
- Java 17
- Maven 3.8+
- Docker
- PostgreSQL

## Démarrer les services
`start-services.bat`
